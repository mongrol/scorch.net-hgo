VPS gone
  • October 30, 2013
  • The Skreegs  
  • raspberrypi  
I’ve now knocked up my other Raspberrypi Model B as a web server to replace my slowing VPS. It’s performing way better than I expected. I’ve set it up with Lighttpd, Postgresql and enabled php-apc to cache php. Php-apc is giving a reasonable speedup of around 30%. The Pi is serving tt-rss external over https, using a server cert from cacert.org. It’s also running dokuwiki which basically gets no action.
